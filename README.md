# jsepoch

> Print UNIX epoch-style timestamp but w/millisecond precision, like JavaScript likes to do

## Table of Contents
- [Install](#install)
- [Usage](#usage)
- [Contributing](#contributing)
- [License](#license)

## Install

```sh
go get -u github.com/nmarley/jsepoch
```

## Usage

Example:

```sh
jsepoch

1575213173522
```
